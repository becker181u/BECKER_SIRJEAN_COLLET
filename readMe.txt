Pour que le site foctionne, il faut créer une base de données appelée "bdd" comportenant deux tables "evenements" et "membres"
La table "evenements" contient les champs ci-après dans l'ordre: "Titre","Description","Date","Infos","Auteur" tous sont des chaînes de caractrères
La table "membres" contient les champs ci-aprés dans l'ordre: "identifiant","Pass","Nom","nbePlaces","Adresse","Ville","CodePostal","Fonction","Mail","Téléphone","Description"
Tous sont des châines de caractères.
Le site de la borne se lance à partir de index.php et le site hébergeur à partir de ProjetInnovant\index.php
